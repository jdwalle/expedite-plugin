<!-- The checkpoint.yml schema that all writes must conform to (from hooks/lib/schemas/checkpoint.js) -->

```yaml
# All fields nullable. requiredWhenPopulated: skill, step, label (if any is set, all three must be)
skill: string | null       # e.g. "scope", "research", "design", "plan", "spike", "execute"
step: number | string | null  # positive integer or "complete"
label: string | null       # human-readable step name

```

3. Set `current_step` to `{skill: "scope", step: 1, label: "Lifecycle Check"}`
