<task id="2" title="INT-02: Align gate outcome schema comment in state.yml.template">
<description>
Single-line edit to the gate_history outcome schema comment.

In `templates/state.yml.template`, locate line 56:

```yaml
#   outcome: "go"            # go | go_advisory | recycle | override
```

Replace with:

```yaml
#   outcome: "go"            # go | go_advisory | hold | recycle | override
```

This adds `hold` between `go_advisory` and `recycle`. The semantic distinction is intentional:
- `hold` = interactive inline fix (scope G1 only, user is present)
- `recycle` = re-run the producing skill (research G2, design G3 -- batch processes)

Do NOT rename `hold` to `recycle` in scope SKILL.md -- the schema comment should be broadened to document all valid outcomes, not the skill narrowed.
</description>
