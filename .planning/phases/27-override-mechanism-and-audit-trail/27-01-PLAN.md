Exports:
- `recordDenial(expediteDir, pattern)` -- Reads `.expedite/.denial-counts.json`, increments count for the given pattern string, writes back. Returns the new count (number). Creates file if missing. Wraps all I/O in try/catch; returns 1 on any error (fail-open).

- `ESCALATION_THRESHOLD` -- constant `3`.

Pattern strings will be formatted by the caller as: `'gate:' + gateId + ':' + targetPhase` for gate denials, `'fsm:' + fromPhase + ':' + toPhase` for FSM denials.
